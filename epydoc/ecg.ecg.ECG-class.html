plot(self, layoutid, mm_mv)

source code 

Plot the ecg signals inside the plotting area. Three possible layout choice are: * 12X1 (one signal per line) * 3X4 (4 signal chunk per line) * 3X4_1 (4 signal chunk per line. on the last line is drawn a complete signal) * ... and much much more

The general rule is that the layout list is formed by as much lists as the number of lines we want to draw into the plotting area, each one containing the number of the signal chunk we want to plot in that line.

Parameters:
  • layoutid (list of list) - the desired layout
